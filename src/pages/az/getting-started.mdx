<h2>Astro nədir?</h2>

Astro **sürətli,** **kontent-fokuslu** vebsaytlar yaratmaq üçün **hamısı birində** **veb-freymvorkdur.**

## Əsas xüsusiyyətlər

- **Komponent Adası:** Daha sürətli vebsaytlar üçün yeni veb arxitekturu.
- **Server öncəlikli API dizaynı:** İstifadəçilərinizin cihazlarından hidrasiyaları çıxardın.
- **Default olaraq Sıfır JS:** Artıq sizi yavaşladan həddən artıq JavaScript runtime yoxdur.
- **Edge-ready:** İstədiyiniz yerə deploy edin, hətta qlobal edge runtimelarından olan Deno və Cloudflare-ə belə.
- **Özəlləşdirilə bilən:** Tailwind, MDX və seçə biləcəyiniz 100-dən çox inteqrasiya.
- **UI-agnostic:** React, Preact, Svelte, Vue, Solid, Lit və daha çoxunu dəstəkləyir.

Astronu nəyin xüsusi etdiyini öyrənmək üçün [Niyə məhz Astro](/az/concepts/why-astro/) bölməsinə baxın. ✨

## İlk layihənizə başlayın

 Bizim `create-astro` CLI komandası ilə tam-hazır Astro layihəsi yaradın!
